# Responsive Portfolio

## Description:

Using Bootstrap CSS Framework to create a mobile responsive portfolio. 

* Create index.html, portfolio.html and contact.html.

* Crecreate your portfolio site with the following items:

* A navbar

* A responsive layout

* Responsive images
